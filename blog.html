Question-2: What is the difference between innerHTML, innerText, and textContent?

innerText: Only the visible text.

innerHTML: Full element including commented text.

textContent: It doesn't interpret html. It provides everything in plain text.

Question-3: What is event delegation in the DOM?
In JavaScript event delegation is a technique to add a single event listener to a parent element instead of multiple child elements. This listener then handles events triggered by its child elements using event bubbling or propagation.
Question-4: What is event bubbling in the DOM?
Event bubbling is a technique that allows events to move up the DOM tree. It makes event handling flexible.
Question-5: How do you create, add, and remove elements using JavaScript?
